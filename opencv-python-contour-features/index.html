目标

练习

  2. （选做）用形状匹配比较两个字母或数字（这相当于很简单的一个OCR噢）。

小结

常用的轮廓特征：

  • cv2.contourArea()算面积，cv2.arcLength()算周长，cv2.boundingRect()算外接矩。
  • cv2.minAreaRect()算最小外接矩，cv2.minEnclosingCircle()算最小外接圆。
  • cv2.matchShapes()进行形状匹配。
