教程

亮度与对比度

亮度调整是将图像像素的强度整体变大/变小，对比度调整指的是图像暗处的像素强度变低，亮出的变高，从而拓宽某个区域内的显示精度。

OpenCV中亮度和对比度应用这个公式来计算：g(x) = αf(x) + β，其中：α(>0)、β常称为增益与偏置值，分别控制图片的亮度和对比度。

经验之谈：此处对α/β控制对比度和亮度有争议，具体请参考：OpenCV关于对比度和亮度的误解

还记得上一节numpy中数据溢出的取模处理吗？250+10 = 260 => 260%256=4，它并不适用于我们的图像处理，所以用np.clip()函数将数据限定：a<0 => a=0, a>255 => a=255

小结

对比度α/亮度β，通过公式g(x) = αf(x) + β来计算。
